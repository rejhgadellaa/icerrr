
RG Thumb Pro // Rejh.nl
A PHP Thumbnail Processor

Version 0.02 (2010.09.13)

=================================================

Use:

rgthumb.php?f=[imgfile]&w=[width]&h=[height]

Optional:

&zc=1			(Zoom-Crop)
&nocache=1		(Disables cache)

&lq=1			(Sets low-quality resizing, makes processing faster)

&jpg_quality=[1-100]	(Sets JPEG-quality, default=100)
&png_compr=[0-10]	(Sets PNG-compression, default=0)
